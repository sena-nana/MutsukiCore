# Mutsuki v0.3.2 阶段报告

## 方向

v0.3.2 承接 v0.3.1 已经补齐的 trace span 发射能力，把 observability 的
JSONL 写入链路补成可读、可验证、可复用的 trace replay contract kit。

## 完成项

- 新增 `JsonlTraceReader`，按 `JsonlTraceWriter` 的 JSONL 格式读回 `TraceSpan`。
- 新增 `mutsuki.testing.replay_trace_spans(...)` 与 `TraceReplayFrame`，用于测试中
  校验 trace 因果链并生成确定性 replay frame。
- trace replay 校验覆盖重复 span、非法时间区间、父链环与可选父链闭包。
- 新增结构化错误码：`trace.record_invalid` 与 `trace.replay_failed`。
- observability/testing 包导出 reader、writer、replay error 与 replay helper。

uv run pytest tests -q
