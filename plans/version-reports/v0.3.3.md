# MutsukiBot v0.3.3 阶段报告

## 方向

v0.3.3 承接 v0.3 MVP 中“确定性目标选择”的后续演进，把固定
`priority + agent_id` 排序提升为可替换的 Agent election policy，同时保持
Agent 生命周期与 `accepts` 过滤由 core 固定执行。

## 完成项

- 新增 `mutsukibot.core.agent_election.AgentElectionPolicy` 协议。
- 新增默认 `PriorityThenIdElectionPolicy`，保持既有排序语义不变。
- `AgentRegistry.install_election_policy(policy, owner=...)` 支持插件安装策略并返回幂等 disposer。
- 策略安装采用栈语义：后安装者优先，卸载后恢复上一策略，全部卸载后恢复默认策略。
- 插件可通过 `PluginScope.add_dispose(...)` 自动恢复策略，满足无副作用热重载。
