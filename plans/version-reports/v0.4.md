# MutsukiCore v0.4 阶段报告

## 方向

v0.4 承接已经完成的 trace replay 闭环，把可复用的契约测试能力收束成稳定
kit，同时把跨 Agent 的因果 trace 断言纳入统一 helper，减少各类测试里的重复
手写断言。

## 完成项

- 新增 `mutsukicore.testing.contract_kit`。
- `assert_trace_tree_closed(...)` 作为 trace replay 的稳定封装，显式要求父链闭合。
- `assert_cross_agent_trace_chain(...)` 将跨 Agent caller/target 的 trace 关系收束成
  单一断言，复用既有 replay 结果。
- `assert_dispatcher_clean(...)` 提供 dispatcher 残留注册的统一断言。
- `tests/core/test_dispatcher_cross_agent.py` 直接复用 kit，证明 contract 断言可以
  从观察性测试中抽出来重用。
